Principais formatações de texto.

Negrito / Destaque

Nesa frase, tenho termo em destaque usamos a tag strong (semantica).

Itálico / Ênfase

Nesta frase vamos transformar essa frase em itálico, Prazer, Drews fizemos isso usando a tag i (não semantica).

Nesta frase, temos um termo em ênfase usando a tag em (semantica).

As tag i e b são obsoletas, não use. Caso queira deixar um texto forte com ênfase use as tag strong ou em.

Texto marcado

Podemos criar também um texto marcado usando a tag MARK.

Texto deletado

Podemos marcar um texto excluido usando a tag del.

Texto pequeno

Podemos deixar o texto pequeno usando a tag small.

Texto grande (obsoleta).

Podemos deixar o texto grande usando a tag (obsoleta) big.

Texto inserido

Podemos inserir o texto usando a tag ins (semantica).

existe também o sublinhado que é a tag u. (não semantica).

Texto sobrescrito

Para inserir coisas do tipo x20+3 usando a tag sup.

Texto subscrito

Para inserir coisas do tipo H2O usando a tag sub.
